About NwSSU Gym Tracker

The NwSSU Gym Tracker is a digital gym management system developed to streamline and modernize how gym operations are handled within Northwest Samar State University. It centralizes gym-related data, simplifies administrative tasks, and provides insightful analytics for better decision-making.

Development Phase

The system was developed as part of our BSIT project under the DevCore Team. It went through several stages:

  • Phase 1: Planning & Requirement Gathering — defining gym needs and workflow.
  • Phase 2: System Design — creating UI mockups and database structure.
  • Phase 3: Backend Development — building core functionalities using Flask and SQLAlchemy.
  • Phase 4: Frontend Integration — developing an interactive and responsive dashboard.
  • Phase 5: Testing & Deployment — ensuring reliability, security, and user experience.

System Overview

The NwSSU Gym Tracker currently focuses on the admin side but will expand to support both members and gym staff. It allows administrators to:

Member Management

Add, edit, or remove gym members with detailed record tracking.

Analytics Dashboard

Monitor revenue, membership trends, and activity logs in real time.

Membership Monitoring

Track expiration dates and log renewals automatically.

Secure Data Storage

All records are stored safely in the database, with proper access control.

Future updates will include a member portal where users can view their workout history, membership progress, and schedule appointments.

Our Goals

Automate manual gym operations to improve efficiency.

Provide visual analytics for gym performance and member data.

Ensure data privacy and secure handling of all user records.

Enhance user experience through responsive, intuitive design.
